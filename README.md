# creative_clock

## Setup & Running

### Xcode

1. Open the project folder in Xcode
2. Select the target scheme from the toolbar
3. Press `Cmd + R` to build and run

#### Prerequisites

Before running the Flutter app, ensure you have the following installed:

- Flutter SDK (latest stable version)
- [PDF](https://fluttermapp.com/static/media/install_flutter_mac.b9b08121d7d36664c2ae.pdf) (Follow this guide to install flutter on macOS)
- Dart SDK (included with Flutter)
- Xcode Command Line Tools (for macOS/iOS development)
  ```bash
  xcode-select --install
  ```
- Android Studio or Android SDK (for Android development)
- CocoaPods (for iOS dependency management)
  ```bash
  sudo gem install cocoapods
  ```

#### Running the Flutter App

**Option 1: Using Flutter Command** 4. Or go to **Product** → **Run**

### VS Code

1. Open the project folder: `code /e:/Applications/creative_clock`
2. Install required extensions (if applicable)
3. Open terminal: `Ctrl + ~`
4. Run build command:
   ```bash
   flutter clean
   flutter pub get
   flutter run --profile
   ```

### General Prerequisites

- Latest Xcode/VS Code installed
- Required dependencies installed
- Node.js/npm (if applicable)
